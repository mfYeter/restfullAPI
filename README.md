# RESTFULL API

Basit bir RESTful API Uygulaması

## Teknolojiler

PHP ve Mysql

## Dosya Dizini

```shell
ideasoft_case/
├───── config/
│  ├───────── connect.php
│  ├───────── function.php
│  ├───────── features

│   ├───────── api_orders.php
│   ├───────── api_products.php
├───── .htaccess 
├───── client.php
├───── ideasoft_case.sql 
└── ...
```

## Installation

Tototo Do is a dockerized application. You can use docker-compose to build up all project. 

It uses a MySQL database that located in a container.
